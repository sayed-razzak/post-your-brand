# 🚀 Post Your Brand

A full-stack web application for managing social media marketing services using AI-generated content.

## 🌐 Live Features

- ✨ AI Content Marketing Services
- 📱 Responsive Website (Mobile + Desktop)
- 📩 Contact Form (Connected with Backend)
- 📧 Email Notification System
- 📊 Admin Dashboard (Django)
- 💬 WhatsApp Integration
- 📸 Instagram Link

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- HTML, CSS

### Backend
- Django
- Django REST Framework

### Database
- SQLite

---

## ⚙️ How It Works

1. User fills contact form
2. Data is sent to Django backend
3. Data is stored in database
4. Email notification is sent
5. Admin can view all leads
